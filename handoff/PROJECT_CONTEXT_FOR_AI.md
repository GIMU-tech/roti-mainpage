# PROJECT_CONTEXT_FOR_AI.md

## 당신이 이해해야 할 프로젝트 정의
이 프로젝트는 **ROTI 브랜드 포털 메인페이지** 구현 프로젝트다.

ROTI는 단순 쇼핑몰이 아니라, 아래 3개 브랜드를 운영하는 전문 브랜드 그룹처럼 보여야 한다.
- ROTI CAMP: 캠핑 / 야외 / 이동 / 휴식
- ROTI HOMESYS: 정리 / 이동 / 수납 / 생활 편의
- LEEL: 주방 / 리빙 / 정돈 / 깨끗함

## 현재까지 합의된 핵심 사항
1. 메인페이지는 브랜드 그룹 포털이어야 한다.
2. 첫 화면은 3장의 브랜드 카드가 중심이다.
3. 카드 전환은 화살표가 아니라 카드 클릭이다.
4. 클릭된 카드는 3D로 회전하며 중앙으로 온다.
5. Hero 아래에서는 한 화면당 한 브랜드가 풀스크린으로 나타난다.
6. 포인트 컬러는 그린이 아니라 레드다.
7. 우측 하단에 있던 메뉴 텍스트는 상단 우측 헤더 영역으로 이동했다.
8. 시안 이미지와 Design.md를 같이 참고해야 한다.

## 작업 우선순위
1. 구조와 컴포넌트 분리
2. Hero 디자인 완성
3. 카드 클릭 인터랙션 구현
4. Hero → 브랜드 풀스크린 스크롤 전환 구현
5. ROTI Group / Final CTA 추가
6. 반응형 / 접근성 / 성능 보정
