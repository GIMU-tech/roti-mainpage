# CODEX_SETUP_BRIEF.md

## 이 자료를 Codex 세션에 넘길 때의 목적
Codex가 단순 코드 생성이 아니라, **기획 + 디자인 규칙 + 구현 구조 + 단계별 제작 순서**를 한 번에 이해하고 작업하도록 만드는 것이 목적이다.

## Codex에게 먼저 전달할 것
1. `Design.md`
2. `PROJECT_CONTEXT_FOR_AI.md`
3. `01_File-Generation-Plan.md`
4. `02_Project-Structure.md`
5. `references/` 폴더 이미지
6. `prompts/00_Global-Rules.md`

## Codex가 지켜야 할 핵심 규칙
- 작업 시작 전 `Design.md`를 읽을 것
- 메인페이지를 쇼핑몰이 아닌 브랜드 포털로 구현할 것
- 화살표 캐러셀 금지
- 카드 클릭 회전 인터랙션 구현
- 스크롤당 한 브랜드 섹션 전환 유지
- 레드 포인트의 절제된 사용 유지

## 추천 Codex 작업 흐름
### 1단계
프로젝트 구조 생성 + 기본 페이지 골격 생성
### 2단계
Hero와 Header의 정적 UI 완성
### 3단계
브랜드 카드 3D 클릭 인터랙션 구현
### 4단계
브랜드 풀스크린 섹션과 스크롤 전환 구현
### 5단계
ROTI Group / Final CTA / 반응형 / 성능 보정

## Codex에게 매 단계 요구할 출력
- 변경 파일 목록
- 구현 요약
- 남은 작업
- Design.md 준수 점검
- 다음 단계 제안
