# 00_Roadmap.md

## 제작 목표
ROTI 메인페이지를 **브랜드 포털형 인터랙티브 랜딩 페이지**로 단계별 제작한다.

## 단계별 로드맵

### Phase 1. Foundation
1. 프로젝트 스캐폴드 생성
2. 기본 폴더 구조 생성
3. 공통 디자인 토큰 / 스타일 변수 생성
4. 브랜드 데이터 구조 생성

### Phase 2. Static UI
5. 고정 헤더 제작
6. Hero 레이아웃 제작
7. 3개 브랜드 카드 정적 배치
8. 우측 상단 네비게이션 배치

### Phase 3. Hero Interaction
9. 카드 hover tilt 구현
10. 카드 클릭 시 중앙 전환 구현
11. 카드 회전/깊이감/선택 상태 구현
12. Hero 상태와 브랜드 데이터 연동

### Phase 4. Scroll Experience
13. Hero pin / scroll transition 구현
14. 카드 → 풀스크린 브랜드 섹션 전환 구현
15. 3개 브랜드 풀스크린 섹션 구현
16. 스크롤 스냅 / 순차 등장 정리

### Phase 5. Brand Portal Completion
17. ROTI Group 소개 섹션 구현
18. Final CTA 구현
19. 쇼핑몰 링크 연결 지점 정리

### Phase 6. Polish
20. 반응형 보정
21. 성능 최적화
22. 접근성 점검
23. QA 및 Design.md 기준 검수

## 산출물 기준
- 작동하는 인터랙티브 메인페이지
- 명확한 컴포넌트 구조
- Design.md
- 변경 대응 가능한 프롬프트 패키지
