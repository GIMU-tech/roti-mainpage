# 03_3D-Card-Interaction.md

## Prompt

`Design.md`를 먼저 읽고 시작하라.

이번 단계에서는 Hero의 핵심 인터랙션인 **브랜드 카드 클릭 전환**을 구현하라.
중요한 점은, 화살표 기반 슬라이드가 아니라 **카드를 직접 클릭하면 해당 카드가 회전하며 중앙으로 오는 구조**여야 한다.

### 목표
- 카드 클릭 기반 브랜드 선택
- 3D turntable / carousel 느낌
- 중앙 선택 카드 상태 반영

### 반드시 구현할 내용
1. 현재 선택된 브랜드 state를 만든다.
2. 카드를 클릭하면:
   - 클릭한 카드가 중앙으로 이동
   - 카드가 회전하며 전면으로 온다
   - 이전 중앙 카드는 좌우 후면으로 물러난다
3. 각 카드에 아래 시각 상태를 준다.
   - active: scale up, opacity 1, brightness 높음, red glow on
   - inactive: scale down, opacity 낮음, rotateY 적용
4. hover 시 미세한 tilt를 준다.
5. 좌측의 index 텍스트도 현재 브랜드에 맞게 갱신한다.
6. 카드 클릭 시 과도한 bounce 없이 premium한 easing을 사용한다.

### 기술 방향
- CSS 3D transform + GSAP 또는 Framer Motion 사용 가능
- 유지보수성을 우선한다
- 모든 카드 위치는 데이터 기반으로 계산 가능하게 하라

### 금지 사항
- 단순 translateX만으로 끝내지 마라
- 화살표 버튼으로 전환하지 마라
- 캐러셀 라이브러리를 가져와 억지로 맞추지 마라

### 완료 기준
- 사용자가 카드를 눌렀을 때 ‘돌아서 앞으로 나온다’는 느낌이 분명해야 한다.
- 시각적으로 중심 카드 / 주변 카드 위계가 분명해야 한다.
