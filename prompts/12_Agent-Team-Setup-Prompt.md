## Prompt

ROTI 브랜드 포털 메인페이지 프로젝트를 **하네스 엔지니어링 + 에이전트 팀** 방식으로 수행하려고 한다.
아래 자료를 읽고, 이 프로젝트에 최적화된 에이전트 팀 구조와 실행 플로우를 설계하라.

필수 참고 자료:
- `Design.md`
- `03_Homepage-Planning-Summary.md`
- `handoff/PROJECT_CONTEXT_FOR_AI.md`
- `handoff/CODEX_SETUP_BRIEF.md`
- `handoff/AGENT_TEAM_AND_HARNESS_PLAN.md`
- `references/` 폴더 이미지들

### 요청사항
1. 이 프로젝트에 필요한 에이전트 역할을 정의하라.
2. 각 에이전트의 입력 자료, 출력 형식, 책임 범위를 정리하라.
3. 단계별 하네스 플로우를 설계하라.
4. 중간 검수 체크포인트를 제안하라.
5. Codex 또는 유사 코드 에이전트 환경에서 운영하기 좋은 형태의 실행 순서를 제시하라.
6. 디자인 일관성을 유지하기 위해 Design.md를 어떤 방식으로 항상 참조하게 할지 제안하라.

2. Harness Flow
