# 05_ROTI-Group-and-CTA.md

## Prompt

`Design.md`를 먼저 읽고 시작하라.

이번 단계에서는 브랜드 풀스크린 섹션 뒤에 이어질 **ROTI Group 소개 섹션**과 **Final CTA**를 구현하라.

### 목표
- ROTI가 단순 쇼핑몰이 아니라 브랜드 그룹이라는 메시지 강화
- 세 브랜드로 이동하는 마지막 진입점 제공

### 반드시 구현할 내용
1. ROTI Group Section:
   - ROTI의 정체성을 짧게 설명
   - 예시 메시지: “ROTI는 일상에서 자주 마주치는 불편함을 더 단순하고 실용적인 제품으로 바꾸는 브랜드 그룹입니다.”
   - 핵심 키워드: MOVE / ORGANIZE / OUTDOOR / KITCHEN
2. Final CTA:
   - 타이틀: “Choose your ROTI brand.” 또는 이에 준하는 문구
   - 3개 브랜드 CTA 버튼 또는 카드 배치
   - 각 버튼은 해당 쇼핑몰 링크 포인트를 갖게 한다
3. 섹션 톤은 앞선 dark premium 시스템을 유지한다.

### 금지 사항
- 갑자기 밝은 기업소개 페이지처럼 바꾸지 마라
- 텍스트를 길게 늘이지 마라
- 상단 Hero와 동떨어진 UI 스타일로 만들지 마라

### 완료 기준
- 페이지 하단까지 내려왔을 때 ‘ROTI 브랜드 포털’이라는 인식이 더 강해져야 한다.
