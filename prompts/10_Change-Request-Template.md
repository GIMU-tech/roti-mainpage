## Prompt Template

`Design.md`를 먼저 읽고 시작하라.

ROTI 브랜드 포털 메인페이지의 일부를 수정하려고 한다.
이번 변경은 기존 디자인 일관성을 유지한 상태에서 진행되어야 한다.

### 변경 요청
[여기에 수정 요청 내용을 구체적으로 작성]

### 반드시 지킬 것
1. `Design.md`를 최우선 기준으로 삼아라.
2. 기존의 dark premium 톤을 유지하라.
3. Hero의 카드 클릭 회전 구조를 해치지 마라.
4. 스크롤 one-brand-per-screen 구조를 해치지 마라.
5. 레드 포인트 사용 원칙을 유지하라.
6. 요청과 무관한 부분은 바꾸지 마라.

### 응답 형식
1. 변경 요약
2. 수정 파일 목록
3. 구현 상세
4. 기존 일관성과의 정합성 설명
5. `Design.md` 준수 체크리스트

## Example Use
- 헤더 높이를 조금 줄이고 싶다
- LEEL 섹션의 CTA 위치를 약간 조정하고 싶다
- Hero 카드 크기를 데스크톱에서 조금 키우고 싶다
- 모바일에서 카드 간격을 더 넓히고 싶다
