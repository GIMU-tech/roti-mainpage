# 06_Animation-and-Polish.md

## Prompt

`Design.md`를 먼저 읽고 시작하라.

이번 단계에서는 전체 메인페이지의 **모션 퀄리티와 연결감**을 다듬어라.
기능 추가보다 완성도 보정이 목적이다.

### 목표
- Hero / Brand Sections / CTA 전체 모션 문법 통일
- red accent와 lighting의 절제된 사용
- premium한 리듬 정리

### 반드시 보정할 내용
1. Hero 최초 진입 애니메이션
2. 카드 hover / click easing 보정
3. Hero → section 전환 타이밍 보정
4. section 텍스트 reveal의 리듬 통일
5. 버튼 hover 상태 통일
6. section 간 전환감 정리
7. 과도한 glow, blur, shadow가 있으면 줄이기

### 품질 기준
- 과시적인 모션이 아니라 고급스럽고 절제된 모션
- 빠르기보다 부드럽고 안정적인 흐름
- dark UI에서 텍스트 가독성 유지

### 완료 기준
- 전체 페이지가 하나의 브랜드 시스템처럼 느껴져야 한다.
