# 07_Mobile-and-Responsive.md

## Prompt

`Design.md`를 먼저 읽고 시작하라.

이번 단계에서는 ROTI 메인페이지를 **태블릿/모바일에서도 자연스럽게 보이도록 반응형 보정**하라.

### 목표
- 데스크톱의 컨셉을 유지하면서 모바일에서도 구조 붕괴 없이 작동
- 카드 인터랙션과 섹션 위계 유지

### 반드시 구현할 내용
1. breakpoint 전략 정리
2. 모바일에서 Header 레이아웃 보정
3. Hero 카드 크기/배치 보정
4. 카드 클릭 인터랙션이 터치 환경에서도 자연스럽게 작동하게 처리
5. 각 브랜드 풀스크린 섹션의 텍스트 크기/간격 보정
6. CTA 가독성 보정
7. 필요하다면 모바일에서는 일부 과도한 3D 효과를 완화

### 금지 사항
- 모바일에서 데스크톱과 완전히 다른 브랜드 경험으로 바꾸지 마라
- 정보 위계를 무너뜨리지 마라

### 완료 기준
- 주요 breakpoint에서 구조가 깨지지 않아야 한다.
