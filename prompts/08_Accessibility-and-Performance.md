# 08_Accessibility-and-Performance.md

## Prompt

`Design.md`를 먼저 읽고 시작하라.

이번 단계에서는 메인페이지의 **접근성과 성능**을 점검하고 개선하라.

### 목표
- 모션 페이지이지만 실제 서비스로 사용할 수 있는 수준의 기본 접근성과 성능 확보

### 반드시 점검/개선할 내용
1. 이미지 lazy loading
2. 큰 배경 이미지 최적화
3. 불필요한 re-render 감소
4. reduced-motion 환경 대응
5. 키보드 포커스 이동 가능 여부 확인
6. 클릭 가능한 카드에 적절한 semantic / aria 처리
7. 텍스트 대비 점검

### 완료 기준
- 무거운 느낌을 줄이고, 기본적인 접근성 이슈가 최소화되어야 한다.
