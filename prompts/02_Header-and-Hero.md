# 02_Header-and-Hero.md

## Prompt

`Design.md`를 먼저 읽고 시작하라.

이번 단계에서는 **Header와 Hero 첫 화면의 정적 완성도**를 높여라.
아직 카드 클릭 회전은 다음 단계에서 구현해도 되지만, 현재 화면만 봐도 시안에 가까운 고급스러운 dark premium 분위기가 느껴져야 한다.

### 목표
- 상단 고정 Header 구현
- Hero 영역 시각 디자인 완성도 향상
- 3개의 브랜드 카드가 중앙 무대처럼 배치되도록 구성

### 반드시 구현할 내용
1. Header:
   - 좌측 상단에 ROTI 로고
   - 우측 상단에 BRAND / NEWS / CAREERS / CONTACT / menu icon
   - 헤더는 fixed
2. Hero 텍스트:
   - 중앙 타이틀: “일상을 위한 세 가지 방식”
   - 서브타이틀: “이동, 정리, 휴식을 위한 서로 다른 해석”
3. 좌측 보조 정보:
   - BRAND PORTAL
   - 현재 인덱스 예: 01 / 03
4. 하단 스크롤 인디케이터 배치
5. 3개 카드 배치:
   - 중앙 카드가 가장 크고 선명
   - 좌우 카드는 어둡고 뒤로 물러난 상태
   - black/charcoal 배경, subtle red accent 사용
6. 카드 디자인:
   - glass/acrylic border 느낌
   - 카드 내부 이미지 + dark overlay
   - 브랜드명 + 한 줄 설명 + red line

### 금지 사항
- 좌우 화살표 추가 금지
- 밝은 회색/화이트 중심 배경으로 변경 금지
- 그린 포인트 사용 금지

### 완료 기준
- 스크린샷만 봐도 지금까지 기획한 메인 시안 방향이 보일 것
- Hero가 브랜드 포털처럼 보여야 하며 쇼핑몰처럼 보이면 안 됨
