# 09_Final-QA.md

## Prompt

`Design.md`를 먼저 읽고 시작하라.

이번 단계에서는 ROTI 메인페이지를 최종 점검하라.
기능 추가보다 QA와 일관성 검토가 목적이다.

### 최종 점검 항목
1. Hero가 브랜드 포털처럼 보이는가?
2. 좌우 화살표가 완전히 제거되었는가?
3. 카드 클릭 시 회전하며 중앙으로 오는가?
4. 스크롤 시 한 화면당 한 브랜드가 분명히 나타나는가?
5. ROTI CAMP / HOMESYS / LEEL의 분위기 차이가 충분한가?
6. 헤더가 항상 상단에서 안정적으로 작동하는가?
7. 레드 포인트가 절제되어 있는가?
8. Final CTA가 자연스럽게 마무리되는가?
9. 모바일/태블릿에서 구조가 깨지지 않는가?
10. Design.md와 충돌하는 요소가 없는가?

### 응답 시 반드시 포함할 것
- QA 결과 요약
- 발견한 문제
- 수정한 파일 목록
- 남아 있는 권장 개선점
- Design.md 준수 여부
